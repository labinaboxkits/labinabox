# Lab in a Box website

A simple static landing page for Lab in a Box.

## Files

- `index.html`
- `styles.css`

## Deploy with GitHub + Netlify

1. Create a new GitHub repository called `lab-in-a-box`.
2. Upload `index.html` and `styles.css` to the repository.
3. Create a Netlify account.
4. In Netlify, choose **Add new site → Import an existing project**.
5. Connect GitHub and select the `lab-in-a-box` repository.
6. Build settings:
   - Build command: leave blank
   - Publish directory: `/` or leave as the default root directory
7. Click **Deploy**.
8. In Netlify, go to **Site configuration → Domain management** to add a custom domain later.

The form uses Netlify Forms. Once deployed, submissions should appear in the Netlify dashboard under Forms.
